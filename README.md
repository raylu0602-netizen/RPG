🗼 《造物主法典：無盡之塔》 (Codex of the Creator: The Endless Tower)

    「在純粹的真理面前，暴力毫無意義。但如果你有無限的真實傷害，外加一副好牌，或許另當別論。」
    
這是一款結合了無限放置增量、硬核數學解謎、以及「大老二」撲克牌局的終極網頁 RPG。玩家將扮演一名踏入維度之塔的探索者，從砍殺史萊姆開始，一路轉生、突破宇宙法則，最終面對數學真理與維度賭徒的挑戰。
✨ 核心特色 (Key Features)

    ⚔️ 指數級的暴力美學：沒有傷害上限。透過無限強化的裝備與轉生倍率，享受傷害數字從幾千一路飆升至幾千兆的純粹爽感。
    🌌 宇宙重構轉生系統：捨棄世俗裝備，換取靈魂昇華。解鎖【✨ 神罰．聖光】與【🌌 風行．維度崩塌】等滅世級真實傷害神技。
    🎲 維度賭徒的牌局：打怪打累了？在特定樓層，Boss 會封鎖你的戰鬥介面，強迫你打一局帶有「出千技能」與「AI 算牌」的大老二 (Big Two)。
    🧮 絕對真理的數學結界：最終 Boss 不吃任何物理與魔法傷害。你必須利用**「分圓多項式 (Cyclotomic Polynomial)」**的代數法則，手動解開 Φ(x) 的方程式才能將其擊敗。
    🎭 機制型爬塔挑戰：每 10 層隨機遭遇四大守衛，無腦放招只會導致滅團，必須活用「燃燒」、「溢補」與「真實傷害」來破局。

🔮 魔法與特殊狀態

    真實傷害：無視敵方防禦力，直接削減生命值。（代表技能：閃電斬、聖光、虛空黑洞）。
    燃燒 (DoT)：依據怪物最大生命值造成百分比傷害，是破解特定護盾的關鍵。溢補機制：治癒術與黑洞吸血可以突破玩家的最大生命值上限，形成黃金護盾。
    風行連擊：啟動後下一次行動（包含攻擊、技能）將發動兩次，且帶有額外倍率。
🛡️ 守衛討伐筆記 (機制 Boss 警告)在爬塔過程中，你將遇到以下四種極度危險的機制守衛：
    
    【絕對屏障 ‧ 鋼鐵巨神兵】：防禦力為無限大 (Infinity)。物理攻擊無效，請準備好你的真實傷害技能。
    【絕對零度 ‧ 冰霜巨龍】：自帶無敵冰盾。必須先使用【火球術】使其進入燃燒狀態，方可造成傷害。
    【猩紅死神 ‧ 塔納托斯】：每 4 回合發動一次造成基於玩家最大生命的真實傷害處決。請善用治癒術進行生命溢補來扛下斬殺。
    【虛空鏡魔 ‧ 迪斯瑪】：會 100% 反射所有真實傷害與魔法。對牠使用大招等於自殺，請回歸初心，使用最純粹的普通攻擊。
🛠️ 安裝與執行 (How to Play)本作採用純前端技術打造，無需安裝任何伺服器環境。

    下載或 Clone 本專案。確保所有圖片資源放置於 images/ 資料夾下。直接使用任何現代瀏覽器（Chrome, Edge, Firefox 等）點擊打開 index.html。開始你的造物主之路。
